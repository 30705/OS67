OS67
===============================
A Simple Operate System.

67 is the magic number :)

####Requirements
* make
* gcc
* binutils
* nasm
* mkfs
* bochs/qemu
     
####Compile  
    ➜  git clone https://github.com/LastAvenger/OS67.git
    ➜  cd OS67
    ➜  make init     # for first time
    ➜  make          
    ➜  make insatll 
    ➜  make run
