OS67
===============================
A Simple Operate System.

67 is the magic number :)

####Requirements
* gcc + nasm + ld + make <br>
* bochs<br>
     
####How to compile  
          git clone https://github.com/LastAvenger/OS67.git
          cd OS67
          mkdir lst
          mkdir bin
          make       # compile
          make run   # run
          make dbg   # debug
