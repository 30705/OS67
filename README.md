OS67
===============================
OS67 is a simple operate system.
67 is the magic number :)

A Simple Operate System.

####Requirements
* gcc + nasm + ld + make <br>
* bochs<br>
     
####How to compile  
          git clone https://github.com/LastAvenger/OS67.git
          mkdir lst
          mkdir bin
          make       # compile
          make run   # run
          make dbg   # debug

####Existing bugs
* when we run it with Virtual Box, fault VINF_EM_TRIPLE_FAULT occured<br>
